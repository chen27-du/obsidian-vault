## 第一天：行业基础入门

### 模块一：煤炭分类与煤质指标（0-32分钟）

- [x] 搞清楚煤炭的工业分类体系（无烟煤、烟煤、褐煤、贫煤等大类的划分依据）
	1.煤产地,所在区域.煤炭埋的深度,煤炭形成的时间
	2.火电厂买煤,本质上就是在效率---同样的钱,烧出更多的电
	3.主要关注烟煤.煤化程度?煤化程度越高,热值越高 越难点燃
- [x] 掌握火电主流用煤——动力烟煤的基本特性（挥发分适中、燃烧充分、热值较高）
	1.核心是动力烟煤,挥发分适中20%-40%,容易点着,燃烧充分.热值中高4500-6000卡,核心产品
	2.褐煤用在煤矿旁边的电厂
- [ ] 理解按**发热量**分类的逻辑（5000大卡、5500大卡、6000大卡对应不同应用场景）
	煤热值如何转化为发电量
- [ ] 理解按**硫分**分类的逻辑（低硫煤 ≤0.5%、中硫煤 0.5%-1.5%、高硫煤 >1.5%）
	- [ ] 烧煤产生二氧化硫,国标35mg/m³,一般标准200,
	- [ ] 硫分越高,生产成本越高
	- [ ] 低硫≤0.5%,0.5%<中低≤1%,1.0%<中≤1.5%,高>1.5%
	- [ ] 成分越高,价格越低,但是脱硫成本增加.并不代表发电成本低
- [ ] 理解按**灰分**分类的逻辑（灰分越低，有效可燃成分越高）
	- [ ] 灰分就是燃烧后的残留物,灰分越低,残留物就越少,处理的成本就越低
	- [ ] 灰分高就是有效燃料少,煤灰处理成本高,灰渣还会加速锅炉磨损
	- [ ] 灰分越低越好,综合成本低
- [ ] 记住：高热值煤 → 降低电厂单位发电成本（每吨煤产出更多电）
- [ ] 记住：低硫煤 → 帮助电厂满足超低排放环保要求（SO₂排放达标）
- [ ] 记住：灰分高 → 增加煤灰处理成本、加速锅炉磨损
- [ ] 理解**挥发分**对锅炉燃烧稳定性的影响
	- [ ] 气态物质占煤重量的百分比
	- [ ] >30%,容易点着,火焰长,但是容易自燃
	- [ ] <15%,难着火,点燃难度高,锅炉不稳定
	- [ ] 20%-35%最理想,着火+稳定
- [ ] 理解**灰熔点**对锅炉结焦风险的影响
	- [ ] 煤灰的软化和熔融温度,关注软化温度ST
	- [ ] 熔点低,煤灰就容易化粘焦化,传热效率-堵住烟气通道-停炉清理-经济停止发电,经济损失
- [ ] 总结：5个关键煤质指标（发热量、硫分、灰分、挥发分、灰熔点）各自对火电企业的具体影响
- [ ] 如何帮客户计算每发一度电的综合成本最低

- [ ] 给到客户的是计算过程和最终答案,而不是单纯的只给出指标

### 模块二：产业链与定价模式（32-70分钟）

- [ ] 梳理产业链第一环：**煤矿开采**（了解矿井类型：井工矿 vs 露天矿，产能规模概念）
	- [ ] 井工矿:地下开采,成本高,安全风险大
	- [ ] 露天矿:地表挖,成本低,产量大
	- [ ] 关注产能,国家有产能核定,旺季不够卖,淡季卖不完
	- [ ] 坑口价是价格链的起点
- [ ] 梳理产业链第二环：**运输中转**（矿区→铁路集运站→港口/直接到厂）
	- [ ] 运费可能比煤贵,同样品质的煤,离电厂越近,越有竞争力
	- [ ] 煤炭中间商流程
- [ ] 梳理产业链第三环：**火电终端**（电厂煤场→输煤系统→锅炉燃烧→发电）
	- [ ] 煤船/煤车到厂 → 过磅称重 → 卸入煤场 → 输煤皮带送入煤仓 → 锅炉燃烧 → 发电
	- [ ] 库存天数(15-30天),太低有断供停机风险,采购部门压力巨大
	- [ ] 太高,占用资金,煤存放久了会氧化降质
	- [ ] 电厂最害怕的事:断供和煤质不稳定
	- [ ] 长期协议:长期供应,年度合同,每月固定量,价格相对稳定
	- [ ] 市场煤,现货采购,临时补货或长期协议之外的需求,价格随行就市

- [ ] 搞清楚**坑口价**的定义：煤矿产地交货价，买方自行承担后续运费
	- [ ] 产地交货,成本+矿方利润+矿方短倒费,买方自己运走
	- [ ] 电厂近的煤矿,大型采购自行运输
- [ ] 搞清楚**平仓价**的定义：港口船上交货价（FOB），包含到港运费但不含海运费
	- [ ] 到港运费,煤炭价格指数,大多指的是平仓价
- [ ] 搞清楚**到厂价**的定义：送到电厂的落地价，包含全部运输费用
	- [ ] 全部运输费用.大多数电厂合同的采购都用到场结算价.
- [ ] 理解三种价格之间的关系：到厂价 = 坑口价 + 运费 + 杂费
	- [ ] 到厂价 = 坑口价 + 铁路运费 + 港口杂费 + 海运费（如有）+ 贸易商加价（如有）
- [ ] 了解坑口价、平仓价、到厂价各自适用的销售场景?
- [ ] 了解碳中和背景下火电行业的定位变化（从基荷电源向调峰电源转型趋势）
- [ ] 理解"火电企业对清洁高效煤需求稳定"的逻辑（即使新能源增长，火电仍承担兜底保供角色）
- [ ] 了解当前火电行业"上大压小"政策（淘汰小机组、建设大容量高效机组）
	- [ ] 核心是产品和综合成本计算能力优势.
采购集中-优质煤,稳定供应,专业服务的销售永远有饭吃

- [ ] 提取本矿煤炭**灰分占比**（如：15%-20%，均值18%）

### 模块二：火电客户核心痛点（35-70分钟）

- [ ] 理解火电企业成本结构：**燃料成本占发电总成本60%以上**，是最大可控成本
- [ ] 梳理痛点①：**控制燃料成本**（直接影响上网电价竞争力和企业利润）
- [ ] 梳理痛点②：**满足环保排放考核**（SO₂、NOx、粉尘排放不达标面临罚款甚至停产）
- [ ] 梳理痛点③：**连续稳定供应**（断供一天可能导致停机，停机一天损失巨大）
- [ ] 梳理痛点④：**煤质稳定性**（煤质波动大会影响锅炉燃烧效率和设备安全）
- [ ] 梳理痛点⑤：**库存管理压力**（存煤过多占资金，过少有断供风险）
- [ ] 弄清火电采购决策链第一个角色：**生产技术部**——关注煤质指标是否匹配锅炉设计参数
- [ ] 弄清火电采购决策链第二个角色：**燃料采购部**——关注价格条款、运输方案、付款条件
- [ ] 弄清火电采购决策链第三个角色：**财务部**——关注付款周期、资金占用成本
- [ ] 弄清火电采购决策链第四个角色：**管理层/分管副总**——关注合作稳定性、综合性价比、战略关系
- [ ] 理解每个角色的**决策权重**和**沟通策略差异**
- [ ] 搞清楚"谁是关键人""谁是影响者""谁是最终拍板者"

### 模块三：客户画像与卖点匹配（70-90分钟）

- [ ] 选定1家本地目标火电客户（优先选有公开信息的大型电厂）

### 模块二：沟通提问法与异议处理（28-65分钟）

- [ ] 掌握"5W1H"提问法框架：Who（谁决策）、What（采购什么煤种）、When（采购周期）、Where（交货地点）、Why（为什么考虑换供应商）、How（怎么结算）
- [ ] 准备提问①：了解客户**年采购量和单批次采购量**
- [ ] 准备提问②：了解客户**采购周期**（月度长协 vs 季度合同 vs 现货采购）
- [ ] 准备提问③：摸清客户**现有供应商的痛点**（价格高？煤质波动？供应不稳？）
- [ ] 准备提问④：确认客户**煤质硬性要求**（热值下限、硫分上限、灰分上限）
- [ ] 准备提问⑤：了解客户**决策流程和关键人**（谁定供应商、审批流程多长）
- [ ] 准备提问⑥：了解客户**运输接收条件**（有无铁路专用线、码头、公路接卸能力）
- [ ] 准备异议回应①：**"你们价格比竞品高"**→回应框架：算单位热值成本，热值高300大卡，每吨发电成本反而低8元
- [ ] 准备异议回应②：**"担心煤质不稳定"**→回应框架：提供近12个月质检数据、承诺到厂复检不合格可退换
- [ ] 准备异议回应③：**"担心供应中断"**→回应框架：说明矿方产能规模、库存保障天数、已服务的稳定客户案例
- [ ] 准备异议回应④：**"我们已经有固定供应商了"**→回应框架：不急于替换，建议先小批量试烧对比，用数据说话
- [ ] 准备异议回应⑤：**"付款条件谈不拢"**→回应框架：灵活设置阶梯付款比例，首批次可适当让步建立信任
- [ ] 每个话术写在卡片或手机备忘录中，便于实战前快速翻阅

## 第四天：合同风险与交割流程

### 模块一：合同五大核心条款（0-33分钟）

- [ ] 学习核心条款①：**质量验收标准**——明确煤质指标的约定值、允许偏差范围
- [ ] 条款①常见陷阱：验收标准写得模糊（如"符合行业标准"而非具体数值）
- [ ] 条款①关键点：明确以哪一方的化验结果为准、争议时如何复检
- [ ] 学习核心条款②：**价格调整机制**——煤价波动时如何调价
- [ ] 条款②常见陷阱：调价触发条件模糊（如"市场价格大幅波动时可协商调价"，什么是"大幅"没定义）
- [ ] 条款②关键点：明确调价基准指数（如环渤海动力煤指数）、调价周期、调价幅度上限
- [ ] 学习核心条款③：**交货时间与数量**——约定月度/季度供应量和交货时间窗口
- [ ] 条款③常见陷阱：交货时间约定过于刚性，未考虑运输延误等不可控因素
- [ ] 条款③关键点：明确"±10%弹性供应量"等合理浮动条款
- [ ] 学习核心条款④：**付款方式**——预付比例、结算周期、付款方式
- [ ] 条款④常见陷阱：付款周期过长（如到厂后90天付款），大量占用卖方资金
- [ ] 条款④关键点：争取"到厂验收合格后30天内付款"或分期付款安排
- [ ] 学习核心条款⑤：**违约责任**——质量不合格、延迟交货、延迟付款的处理
- [ ] 条款⑤常见陷阱：扣款标准不明确，仅写"按实际损失赔偿"
- [ ] 条款⑤关键点：明确违约金计算方式（如每延迟一天按合同金额0.05%计）
- [ ] 整理合同常见陷阱清单：质量异议期短于7天、调价触发条件模糊、扣款标准不明确、付款周期过长、违约责任不对等
- [ ] 理解"质量异议期"的重要性：异议期太短，买方来不及完成复检

### 模块二：运输方式与交割流程（33-70分钟）

- [ ] 了解**公路运输**特点：灵活机动、适合小批量（单次50吨以内）、短距离（300公里以内）、运费最高
- [ ] 了解**铁路运输**特点：成本较低、适合大运量（万吨级以上）、长途（500公里以上）、需铁路计划审批
- [ ] 了解**水路运输**特点：运费最低、适合沿海沿江电厂、运量最大、受航道和季节影响
- [ ] 理解不同运输方式对应的**运费成本区间**差异
- [ ] 了解铁路运输的**请车皮**流程和时效性问题
- [ ] 了解**多式联运**概念（如公路短倒+铁路长途+水路转运的组合方案）
- [ ] 掌握火电交割流程第一步：**到厂抽样**（电厂在煤到厂后按规定方法取样）
- [ ] 掌握火电交割流程第二步：**第三方化验**（双方认可的检测机构出具化验报告）
- [ ] 掌握火电交割流程第三步：**结算**（根据化验结果，按合同约定的质量-价格对应关系结算）
- [ ] 理解**以质论价**机制：热值高于基准值加价，低于基准值扣价
- [ ] 理解煤质争议处理流程：第一步双方协商→第二步申请复检（送第三方权威机构）→第三步仲裁或诉讼
- [ ] 理解火电企业对"连续供应"的刚性要求（断供可能导致停机事故）
- [ ] 了解协调矿方产能保障供应的基本思路：提前锁定产能、建立安全库存、签订长期协议
- [ ] 了解电厂**煤场管理**基本概念（最低库存警戒线、合理库存天数通常15-30天）

### 模块三：合同实操标注（70-90分钟）

- [ ] 找到公司一份成熟火电销售合同（向主管或法务索取）

- [ ] 标注合同中的**价格条款**，确认是否有调价机制及触发条件

- [ ] 标注合同中的**付款条款**，确认结算周期和付款方式
- [ ] 标注合同中的**违约责任条款**，确认扣罚标准是否明确

### 模块一：案例拆解（0-38分钟）

- [ ] 案例①关键动作②：不只报吨价，而是帮客户算"每发一度电的燃料成本"（单位热值成本）
